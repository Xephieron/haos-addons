# Xephieron Home Assistant Add-ons

Ein Repository mit eigenen Add-ons für Home Assistant OS.

## Add-ons hinzufügen

Füge dieses Repository in Home Assistant hinzu:

**Einstellungen → Add-ons → Add-on-Store → ⋮ → Repositories**

```
https://github.com/Xephieron/haos-addons
```

[![Open your Home Assistant instance and show the add add-on repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2FXephieron%2Fhaos-addons)
